About

FreeScience.info is an initiative to create a virtual libriry of scienfic books. Freescience.info version 1.0 was created by Claudio Attaccalite in 2002 and upgraded to version 2.0 in 2019. The number of scientific books freely available on internet is rapidly growing. This is due mainly to two reasons: first people are starting to release books on Creative Commons licence; second more and more of the old scientific books are made available on the web. If you want all us, please report new books here. If you find that some of the books on FreeScience.info violates copyrights low, please contact us.
